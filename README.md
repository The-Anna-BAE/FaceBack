# FaceBack
Система компьютерного зрения на базе глубокого обучения для распознавания человеческих эмоций в режиме реального времени и анализа видеопотоков. Проект объединяет детекцию лиц и классификацию эмоциональных состояний для комплексной оценки реакций аудитории.

## Области применения (Use Cases)

* Анализ фокус-групп и маркетинг: Определение целевой аудитории бренда и оценка эмоционального отклика на новые рекламные кампании, продукты или дизайн упаковки.
* UX/UI Исследования: Тестирование интерфейсов приложений и сайтов (отслеживание моментов фрустрации, удивления или удовлетворения пользователя в процессе взаимодействия).
* EdTech (Образовательные технологии): Оценка вовлеченности и концентрации студентов во время онлайн-лекций или прохождения интерактивных курсов.
* Геймдев (Game Dev): Анализ реакций геймеров во время плейтестов для корректировки сложности и эмоционального темпа игры.

## Установка

1. Клонируйте репозиторий:


git clone https://github.com/your-username/emotion_project.git
cd emotion_project



2. Создайте и активируйте виртуальное окружение:


python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate



3. Установите необходимые зависимости:


pip install -r requirements.txt



> *Примечание: Для работы с ноутбуками убедитесь, что у вас установлен Jupyter (pip install jupyter lab).*

## Использование

Вся логика проекта, включая инференс моделей и интерфейс взаимодействия, сосредоточена в интерактивном ноутбуке undoxing_emotion.ipynb.

### Порядок запуска:

1. Запустите Jupyter Lab или откройте ноутбук в любимой IDE (VS Code / PyCharm / Google Colab).
2. Откройте файл undoxing_emotion.ipynb.
3. Последовательно выполняйте ячейки, начиная с самой первой (Shift + Enter).
4. При запуске основного блока интерактивный скрипт во время работы (внутри ячейки) попросит вас выбрать режим:
* Режим реального времени: Программа активирует вашу веб-камеру для потокового анализа лица.
* Работа с подготовленным файлом: Скрипт попросит ввести полный путь к вашему видеофайлу на диске.



> Важно: Все необходимые предобученные веса и архитектуры моделей подгружаются автоматически из Hugging Face и соответствующих репозиториев при первом запуске ноутбука. Ничего скачивать вручную не нужно.

## Структура проекта

Для визуализации архитектуры решения используется следующая схема:


graph TD
    A[undoxing_emotion.ipynb] --> B{Выбор режима}
    B -->|1. Реальное время| C[Поток с веб-камеры]
    B -->|2. Видеофайл| D[Полный путь к .mp4/.avi]
    C --> E[Детекция лиц: YOLO Hugging Face]
    D --> E
    E --> F[Классификация эмоций: Keras/TensorFlow]
    F --> G[Отрисовка результатов: OpenCV UI]



* undoxing_emotion.ipynb — основной и единственный рабочий Jupyter Notebook проекта.
* requirements.txt — список зависимостей (TensorFlow, OpenCV, Hugging Face библиотеки и др.).

## Данные и Модели

* Детекция лиц: Используется современная архитектура YOLO, оптимизированная для поиска лиц и загружаемая напрямую через API Hugging Face Hub.
* Распознавание эмоций: Глубокая сверточная нейросеть (CNN), реализованная на Keras / TensorFlow, которая классифицирует выражения лиц по базовым категориям (радость, грусть, удивление, гнев, нейтральное и др.).
