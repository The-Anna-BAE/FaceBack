# FaceBack
Система компьютерного зрения на базе глубокого обучения для распознавания человеческих эмоций в режиме реального времени и анализа видеопотоков. Проект объединяет детекцию лиц и классификацию эмоциональных состояний для комплексной оценки реакций аудитории.

## Области применения

* Анализ фокус-групп и маркетинг: Определение целевой аудитории бренда и оценка эмоционального отклика на новые рекламные кампании, продукты или дизайн упаковки.
* UX/UI Исследования: Тестирование интерфейсов приложений и сайтов (отслеживание моментов фрустрации, удивления или удовлетворения пользователя в процессе взаимодействия).
* EdTech: Оценка вовлеченности и концентрации студентов во время онлайн-лекций или прохождения интерактивных курсов.


## Ключевые возможности

* Мультимодальный инференс: поддержка работы как в режиме реального времени (через потоковый захват с веб-камеры), так и в режиме пакетной обработки заранее записанных видеофайлов любого хронометража.
* Высокоточная детекция лиц: использование современной легковесной архитектуры YOLO11-Face, устойчивой к перекрытиям лиц (occlusions), изменениям ракурса и плохим условиям освещения.
* Реал-тайм трекинг метрик: отрисовка на каждом кадре ограничивающей рамки, текстового маркера доминирующей эмоции.
* Автоматическое логирование и экспорт: логирование предсказаний по каждому кадру и автоматическая сборка финальной аналитической таблицы для последующего построения графиков динамики эмоций.


## Стек технологий

* Компьютерное зрение (Computer Vision): OpenCV (захват видеопотока, предобработка кадров, наложение графического интерфейса и сохранение результатов).
* Глубокое обучение (Deep Learning): TensorFlow & Keras (импорт, дообучение и инференс глубокой сверточной нейросети классификации эмоций).
* Предобученные архитектуры: YOLO11-Face (загрузка и использование весов современной модели детекции лиц через экосистему Hugging Face Hub).
* Анализ и структурирование данных: Pandas (формирование таймлайна эмоций, фильтрация логов и экспорт результатов в табличные форматы).


## Установка

1. Клонируйте репозиторий:


git clone https://github.com/The-Anna-BAE/FaceBack.git
cd FaceBack



2. Создайте и активируйте виртуальное окружение:


python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate



3. Установите необходимые зависимости:


pip install -r requirements.txt



> *Примечание: Для работы с ноутбуками убедитесь, что у вас установлен Jupyter (pip install jupyter lab).*

## Использование

Вся логика проекта, включая инференс моделей и интерфейс взаимодействия, сосредоточена в интерактивном ноутбуке undoxing_emotion.ipynb.

### Порядок запуска:

1. Запустите Jupyter Lab или откройте ноутбук в предпочтительной IDE (VS Code / Google Colab / Jupyter Notebook).
2. Откройте файл undoxing_emotion.ipynb.
3. Последовательно выполняйте ячейки, начиная с самой первой (Shift + Enter).
4. При запуске основного блока интерактивный скрипт во время работы (внутри ячейки) попросит вас выбрать режим:
* Режим реального времени: Программа активирует вашу веб-камеру для потокового анализа лица.
* Работа с подготовленным файлом: Скрипт попросит ввести полный путь к вашему видеофайлу на диске.



> Нужно выбрать и записать режим именно **в той формулировке, которая указана**.  
> Важно: Все необходимые предобученные веса и архитектуры моделей подгружаются автоматически при выполнении ячеек кода при запуске ноутбука. Ничего скачивать вручную не нужно.

## Структура проекта

* undoxing_emotion.ipynb — основной и единственный рабочий Jupyter Notebook проекта.
* requirements.txt — список зависимостей (TensorFlow, OpenCV, Hugging Face библиотеки и др.).

## Данные и Модели

* Детекция лиц: Используется модель YOLO11-Face, которая оптимизирована для поиска лиц. Загрузка весов - из Hugging Face Hub.
* Распознавание эмоций: Глубокая сверточная нейросеть (CNN), полученная путен fine-tuning модели ResNet18, которая обучена на датасете FER-2013. Модель классифицирует выражения лиц по базовым категориям (радость, грусть, удивление, гнев, нейтральное и др.).
